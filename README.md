# Nyx 📸

> **Capture together. Share instantly.**

Nyx is a real-time collaborative photo-sharing platform that lets groups instantly share memories as they happen.

Whether you're on a college trip, at a farewell, hackathon, wedding, birthday, or any group event, every photo captured through Nyx is automatically synced to a shared event gallery—so everyone has every moment without asking for photos later.

## ✨ Core Idea

Sharing photos after an event is often slow and fragmented. Pictures end up scattered across different phones, messaging apps, and cloud drives.

Nyx solves this by creating a live, shared space where every participant contributes automatically. Capture a photo, and it appears for everyone in the event within seconds.

## 🚀 Core Features

* 📸 Instant photo sharing
* 👥 Event-based shared spaces
* 🖼️ Live shared gallery with everyone's photos
* 👤 Individual galleries for each participant
* ☁️ Automatic cloud synchronization
* 🔄 Offline upload queue
* 🔒 Private events with invite codes or QR codes

## 🛠️ Tech Stack

* **Android:** Kotlin, Jetpack Compose
* **Backend:** Go, REST APIs, WebSockets
* **Database:** PostgreSQL, Redis
* **Storage:** AWS S3
* **DevOps:** Docker, Kubernetes, Jenkins

## 🎯 Vision

Nyx aims to make group photo sharing effortless. Instead of spending time collecting pictures after an event, everyone enjoys a live, collaborative gallery where every memory is shared instantly.

**One event. One gallery. Every perspective.**
